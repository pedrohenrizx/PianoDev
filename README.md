# **🎹 Virtuoso Piano \- Simulador Digital**

O **Virtuoso Piano** é uma aplicação web interativa que simula um piano funcional diretamente no navegador. Utilizando tecnologias modernas de síntese de áudio, o projeto oferece uma experiência fluida sem a necessidade de ficheiros de som externos (MP3/WAV), gerando ondas sonoras em tempo real.

## **🚀 Tecnologias Utilizadas**

* **HTML5**: Estruturação semântica do simulador.  
* **Tailwind CSS**: Estilização moderna, responsiva e utilitária.  
* **JavaScript (Vanilla)**: Lógica de interação com o DOM.  
* **Web Audio API**: Motor de síntese sonora para geração de frequências em tempo real.

## **✨ Funcionalidades**

* **Síntese de Som em Tempo Real**: Utiliza osciladores para criar timbres de piano sintetizado.  
* **Interatividade Dupla**: Toque utilizando o rato ou o teclado do computador.  
* **Design Responsivo**: Interface que se adapta a diferentes tamanhos de ecrã.  
* **Controlo de Volume**: Ajuste a intensidade do som através de um slider intuitivo.  
* **Feedback Visual**: As teclas iluminam-se e movem-se ao serem premidas, simulando a mecânica de um piano real.

## **🛠️ Como Executar**

Como o projeto foi desenvolvido num único ficheiro autónomo:

1. Abra o ficheiro piano\_virtuoso.html em qualquer navegador moderno (Chrome, Firefox, Edge, Safari).  
2. Interaja com a interface para ativar o contexto de áudio do navegador.  
3. Comece a criar música\!
